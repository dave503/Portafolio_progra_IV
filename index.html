Introducción

En este primer cómputo he adquirido los conocimientos basicos sobre los protocolos y las tecnologías web, también conocí lo básico a cerca de CSS3 que es para darle estilo a las páginas web, además de las etiquetas básicas para poder enmaquetar una página web en HTML5.
Estudié la sintaxis basica del lenguaje de programación PHP, que trabajeré en la versión 7, en este lenguaje he adquirido el conocimiento sobre la declaración de variables, cómo trabajar con ellas, ademas conocí las distintas estructuras de control (FOR, WHILE, DO WHILE y el FOREACH), también las condicionales (IF ELSE y el SWITCH), el funcionamiento y cómo aplicarlas en el código PHP.
También aquirimos habilidades para trabajar con arrays de una y varias dimensiones, también arrays asociativos y como podemos mostrarlos utilizando diferentes estructuras de control.
